# NBA Stats via NBAScraper
A ReactJS app displaying LIVE NBA player statistics.

## Technologies Used

## ReactJS version

Built using React version 16.1.0

## Configuration

    $ git clone https://github.com/edgarsbarajas/nba-stats.git

    $ cd nba-stats

    $ npm install

    $ npm start
